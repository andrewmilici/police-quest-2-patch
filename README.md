## ‚öôÔ∏è How to use

1. You must own an original copy of **Police Quest II**.
2. Download a copy of the patch files (PatchFiles.zip) from the [Releases](https://github.com/andrewmilici/police-quest-2-patch/releases/latest) page.
3. Extract the files in the zip straight into your Police Quest 2 folder. 

**Note** At this stage, the game is best run using ScummVM. While the game will run using Dosbox or other means, you may encounter the odd "out of heap space" error. These may be addressed in the future. ScummVM will likely warn you that it has detected an unknown game variant, this is normal and you should be able to proceed past this by selecting "Add anyway".
